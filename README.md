# Bookshelf

Demo: https://timardex.github.io/Bookshelf/
Cd into frontend and run `npm install` and `npm start`. To run test `npm run test`.

### You can

1. Filter books by:

-   book genre
-   author name
-   author gender
-   book published year

2. Sort books by:

-   book name ASC and DESC
-   author name ASC and DESC
-   favourites

3. Search books by:

-   author
-   book
-   genre

4. Load more books
5. Clear all filters and search
